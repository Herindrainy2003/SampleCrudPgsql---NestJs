## Description
Sample crud for learning about NestJs + TypeOrm + postgresql(all endpoints => get ,post , patch ,delete)

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
